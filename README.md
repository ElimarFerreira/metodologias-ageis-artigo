# Projeto: A Importância das Metodologias Ágeis no Desenvolvimento de Software

Este repositório contém o artigo técnico desenvolvido utilizando:

* Scrum como metodologia de gestão
* Trello como quadro Kanban
* Git e Github para versionamento
* LaTeX (Overleaf) para escrita acadêmica

O projeto segue uma Sprint simulada de 15 dias, com registro diário das atividades.

## **Estrutura**

* /src: arquivos .tex, bibliografia e imagens
* /figuras: diagramas usados no trabalho
* /secoes: texto dividido por capítulos
* /tabelas: conteúdos auxiliares
* main.tex: arquivo principal

## **Branches**

* main – versão estável
* feature/secao-intro
* feature/secao-scrum
* feature/beneficios-ageis
* feature/figura-scrum
* feature/tabela-comparativa
* feature/bibliografia
* feature/expressões-matematicas
* feature/formatação-artigo

## **Objetivo**

Entregar um artigo acadêmico completo atendendo aos requisitos da disciplina.
